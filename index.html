DISC Test

Project On Department

Created by Verryanto Paulus, SD II Indomaret Group

Latar Belakang

  1. Proses input hasil tes DISC masih menggunakan kertas (form fisik)
  2. Tidak adanya integrasi dengan sistem HRIS
  3. Pelamar yang sudah melakukan tes DISC tidak terdokumentasi
  4. Cost yang diperlukan untuk biaya operasional(biaya kertas)

Tujuan

  1. Kemudahan dalam mengakses data pelamar yang sudah melakukan tes secara online
  2. Menurunkan biaya operasional untuk mencetak kertas
  3. Digitalisasi Tes DISC

								== Website Career ==
								user -> career: Apply
								career -> hris: Data masuk ke HRIS
								career -> disc: sinkron ke DISC
								activate disc #A80036
								disc -> career: generate akses token
								note right : Perubahan yang terjadi pada blok MERAH
								career -> user: kirim akses token by email
								user -> disc: user melakukan tes DISC

								disc -> hris: NOK atau OK
								deactivate disc
	
								== HRIS ==
								hris -> screening : Melakukan proses screen CV
								screening -> psikotes: Blasting email dan melakukan psikotes
								psikotes -> dpp: Input DPP
								dpp -> interview: Interview HRD dan User
								
								interview -> pkwt: Proses PKWT
								== Sync Ke HRMS ==
								hris -> cbn
								cbn -> hrms
								@enduml
